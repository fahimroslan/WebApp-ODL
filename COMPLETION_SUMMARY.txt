DATABASE SCHEMA CREATED
═════════════════════════════════════════════════════════════════════════════

user_roles
  - Stores admin/student role assignments
  - Links to Supabase auth.users
  - RLS: Each user can see their own role

students
  - IC (unique identifier)
  - Name and intake code
  - Optional user_id link for registered students
  - RLS: Students see own profile, admins see all

courses
  - Code, title, credits
  - Description field
  - RLS: Authenticated users read, admins write

enrollments
  - Links students to courses
  - Stores marks, grades, GPA data
  - Semester and session information
  - RLS: Students see own enrollments, admins see all

audit_logs
  - Tracks all admin actions
  - User, action, timestamp, changes
  - RLS: Admin-only access
